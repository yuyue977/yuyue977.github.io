# yuyue's Personal Website

[![Deploy](https://img.shields.io/badge/deploy-GitHub%20Pages-blue)](https://yuyue977.github.io)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

个人网站，展示个人介绍、技术栈和项目作品。

## 技术栈

- HTML5 语义化标签
- CSS3 (CSS Variables, Grid, Flexbox, 动画)
- Vanilla JavaScript (ES6+)
- 响应式设计 + 暗色模式

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/yuyue977/yuyue977.github.io.git

# 直接用浏览器打开 index.html，或使用任意静态服务器
npx serve .
```

## 项目结构

```
├── index.html    # 主页面
├── styles.css    # 样式文件（含响应式和暗色模式）
├── main.js       # 交互逻辑（主题切换、滚动动画）
└── README.md
```

## 功能特性

- ✅ 响应式布局，完美适配移动端
- ✅ 暗色/亮色主题切换
- ✅ 平滑滚动动画
- ✅ 语义化 HTML + SEO 优化
- ✅ 纯原生实现，无框架依赖
